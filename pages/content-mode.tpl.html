Modes

Normal Mode

In normal mode (aka app mode) all the margin and padding values are set to 0. That way you have more control over how things should be arranged or spaced, using pre-defined classes like .arrange or .space.

Content Mode

In content mode all the "text" tags will get margin bottom automatically. All you need is the content-mode wrapping class and then all the children will get margin bottoms.
