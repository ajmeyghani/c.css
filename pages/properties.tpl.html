Spaces

Space properties can be used when spacing elements on a page to ensure consistency. The base value is at 1rem, so if the base font size is 16px, the base space value is then computed as 16px. All the space-related properties are listed below:

  • space-half
  • space
  • space-x2
  • space-x3
  • space-x4

Fonts

  • --base-font-size 20px;
  • --body-font-size 0.95rem;
  • --base-font-lh 1.5rem;

Animation Properties

  • --transition-duration 250ms;
  • --transition-easing ease-in-out;
  • --transition-all all var(--transition-duration) var(--transition-easing);
